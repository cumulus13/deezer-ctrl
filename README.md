# Deezer-Ctrl

## Control Deezer on Chrome running with "--remote-debugging-port=?" and PyChrome.

### Requirements (pip)

- pychrome
- pydebugger
- configset
- requests
- beautifulsoup4 (or bs4)
- make_colors
